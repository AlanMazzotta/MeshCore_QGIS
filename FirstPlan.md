# First Plan

This file retains the original project plan and background context for MeshCore_QGIS.

- Purpose: Combine MeshCore network data with QGIS for repeater placement (viewshed analysis).
- Data sources discussed: map.meshcore.dev, MeshCore MQTT bridge, MeshCore Python API, MESH-API.
- High-level workflow: extract node data → load into QGIS → obtain DEM → run viewshed analyses → iterate on repeater placement.
- Notes: Recommendations for UAS photogrammetry for local DSMs, validation with RSSI/SNR telemetry, automation ideas.

Refer to the new `README.md` for the current project status and how to get started.
